Project Financial Summary

Outstanding Invoices
The sum of all the still pending transactions associated with invoices for this project.

Paid Invoices
The sum of all the approved transactions associated with invoices for this project.

Pending Time Sheets
The sum of all the still pending transactions associated with time sheets for this project.

Paid Time Sheets
The sum of all the approved transactions associated with time sheets for this project.

Task Estimate
Tasks for this Project that have been Time Estimated by someone who is on this Project and has a rate set, can be tallied up to determine an estimate of how much cost and time is remaining for all the incomplete Tasks.

Eg: Task Quote: $100.00 / 5.0 Hours (4 incomplete, 2 not included)

There is $100 worth of work left, and it should only take 5 hours. Also, there are 4 tasks yet to be completed and of those, 2 of them cannot be included in this Task Quote because they haven't got Time Estimates filled in.
