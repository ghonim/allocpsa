Add Time Hint Help

This quick entry box enables a manager to log time sheet item hints for particular people and their tasks.

Later these hints can be used to see if a person has billed less than the manager anticipated.

To record that a particular person with the username fred looks to have done two and a half hours of work today for task 1234, input this and hit enter.

fred  2.5  1234  Freddy called the client.


Advanced Usage

The text format can be expanded to include a date, if entering a date other than today (fields surrounded by square brackets are optional)

USERNAME  [DATE]  DURATION  [TASK_ID]  [COMMENT]
