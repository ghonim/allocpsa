Save
The Save button will save the time sheet. It does not save the Time Sheet Line Items below, and it will not take you away from this page.

Delete
This button will erase the current Time Sheet from the system. All the line entries that you have entered will be deleted.

To Manager
You will only see this button here if a Manager has been selected for this particular Project. This button will save the current Time Sheet and submit it to the Manager. You will not be able to edit this Time Sheet any further if you click this button, unless the Project Manager rejects it and decides to make the Time Sheet editable again for some reason.

To Admin
This button will save the current Time Sheet and submit it to the Time Sheet Administrator. You will not be able to edit this Time Sheet any further if you click this button, unless the Time Sheet Administrator rejects it and either sends it back to the Project Manager or simply decides to make it editable again.
