Add Time Help

To record two and a half hours of work today for task 1234, input this and hit enter.

2.5  1234  Called client regarding thing.


Advanced Usage

The text format can be expanded to look like this:
(fields surrounded by square brackets are optional)

[DATE]  DURATION  [UNIT]  [x MULT]  [TASK_ID]  [COMMENT]

Thus, to add a time sheet item for task 1234 for work done on halloween '97, that took one and a half days, that was billed at double time:

1997-10-31  1.5 days  x2  1234  Halloween work.
