Automatically Reopen Task On This Date

If you would like this task to automatically reopen on a particular date, then input the date here.

A new reminder will be created for the date you specify. When that particular reminder fires, the task will change its status to Open: In Progress.

If you wish to cancel this future event, then go into the reminder and deactivate or delete it.

Note that this functionality is dependant on the reminders functionality being enabled.
